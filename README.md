# 🚀 Automated Backup System using AWS EC2

## 📖 Overview
Automated backup system built on AWS EC2 using Bash scripting. It creates backups, uploads to Google Drive using rclone, and schedules jobs using cron.

## ⚙️ Tech Stack
- AWS EC2
- Linux (Ubuntu)
- Bash Scripting
- rclone
- Cron Jobs

## ✨ Features
- Automated backups
- Google Drive upload
- Cron scheduling
- Logging system

## 🚀 How to Run
./backup.sh

## ⏰ Cron Job
crontab -l

## 📊 Logs
cat ~/backup.log
